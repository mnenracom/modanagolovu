# Информация об ошибках в админ-панели

Если админ-панель показывает белый экран, проверьте:

1. **Консоль браузера** (F12 → Console) - там должны быть ошибки JavaScript
2. **Сеть** (F12 → Network) - проверьте, есть ли ошибки при загрузке данных

## Возможные причины:

1. **Таблица settings не создана** - выполните SQL-скрипт `create_settings_table.sql`
2. **Ошибки RLS** - проверьте права доступа в Supabase
3. **Ошибки в компонентах** - проверьте консоль браузера

## Быстрое решение:

Если белый экран, откройте консоль браузера и найдите ошибку. Чаще всего это:
- `Cannot read property 'X' of undefined` - проблема с данными
- `Failed to fetch` - проблема с подключением к Supabase
- `Policy violation` - проблема с RLS
